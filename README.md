# 小型图书管理系统 python1047

## 项目概述

本项目是一款使用Python开发的小型图书管理系统，旨在为图书管理提供一个便捷、高效的管理解决方案。系统分为管理员和普通用户两个角色，分别具备不同的权限和功能。

## 技术栈

- 后端：Python
- 数据库：SQLite
- 前端：HTML、CSS、JavaScript（可选）

## 功能模块

### 管理员

- 图书信息管理
  - 添加图书
  - 修改图书信息
  - 删除图书
- 用户管理
  - 查看用户列表
  - 禁用/启用用户

### 用户

- 查找图书
- 借阅图书
- 归还图书

## 系统角色

- 管理员：负责图书和用户管理
- 用户：进行图书查找、借阅和归还

## 运行环境

- Python版本：3.x
- 操作系统：Windows、Linux或macOS
- 数据库：SQLite

## 部署步骤

1. 克隆或下载项目代码
2. 安装依赖包：`pip install -r requirements.txt`
3. 初始化数据库：运行`initialize.db`脚本
4. 运行项目：`python manage.py runserver`
